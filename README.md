# CxxMidi

CxxMidi is a C++ MIDI library.

## Design
* Multiplatform (Linux, Windows)
* Endian safe
* hpp header-only library

## Key featurs
* MIDI files read/write
* MIDI outputs
  * ALSA on Linux
  * WinMM on Windows
* Multiplatform MIDI players
  * synchronous
  * asyncrhonous
